# 自訂主題

Vnite 的使用者介面由 Shadcn UI 進行建構，充分支援其主題配置方法，您可以在 `設置-主題` 中透過 css 檔案配置您的主題，會不定期更新內置預設可供選擇。

![customTheme2](https://img.timero.xyz/i/2025/04/02/67ed28f963528.webp)
